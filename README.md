# 목장 기록 프로그램

문자열을 입력하면 정해진 정규화를 통해 문자열을 분리하여 db에 저장

ex) 

문자열 입력 시:

[[1번축사,2번축사]] 축사 테스트,

[[1-1,1-2]] 축사 칸 테스트,

[[1111]] 소 테스트 1,

[[2222]] 소 테스트2
